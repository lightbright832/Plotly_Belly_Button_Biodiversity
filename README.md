![](https://github.com/lightbright832/Plotly_Belly_Button_Biodiversity/blob/main/static/images/Customized%20Dashboard.png)

## Overview of Analysis
### Purpose:
The following analysis was completed to showcase how JavaScript can be used to create an interactive dashboard containing information on belly button bacteria. The customizations that can be created with JavaScript, JSON, and Plotly help to create a dynamic dashboard that can fit the needs of many industries.
